# 🏦 Banking Operations Analytics Dashboard

An end-to-end banking data analytics and reporting platform built with Python, SQL, and Machine Learning — simulating real-world operational intelligence workflows used in financial institutions.

## 🔗 Live Dashboard
[View Live App](https://banking-analytics-dashboard-t5dhpgg96vzl2vvjn9rzhf.streamlit.app/)

## 📊 Features

### 1. Transaction Analysis
- Interactive filters by bank, region, transaction type and date
- Volume trends, status breakdowns, heatmaps and day-of-week patterns
- Box plots for amount distribution across banks

### 2. Anomaly Detection (ML-Powered)
- Isolation Forest algorithm to flag suspicious transactions
- Adjustable contamination rate via sidebar
- Anomaly scoring, trend over time, and flagged transaction table

### 3. Regional Performance Report
- K-Means clustering to segment regions into performance tiers
- High Performer / Developing / Needs Attention classification
- Bank distribution by region, failed rate analysis

### 4. Executive Summary
- Risk scoring engine (Low / Medium / High / Critical)
- SLA breach tracking per bank
- Bank performance scorecard with composite scoring
- Volume vs Anomalies dual-axis trend chart
- 3 data-driven policy recommendations for senior leadership

## 🛠️ Tech Stack
- **Python** — Pandas, NumPy, Scikit-learn
- **Visualization** — Plotly, Streamlit
- **ML Models** — Isolation Forest, K-Means Clustering
- **Deployment** — Streamlit Cloud, GitHub CI/CD
